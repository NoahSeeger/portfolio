# Dark Mode Color Palette Notes

## User-Approved Dark Mode Palette (2026-05-20)

用户提供的新 dark mode 配色方案：

```css
/* Dark Mode (proposed) */
--bg-primary: #212737;    /* dark blue-gray */
--accent: #FE6B02;        /* vibrant orange */
--text-primary: #EAEDF3;   /* light white */
--text-secondary: #C2C5CD; /* muted gray */
```

## Feedback on Palette

Orange accent (#FE6B02):
- Bold, vibrant - stands out
- Risk: 10% accent might be too intense with this brightness
- Consider: slightly darker orange like #E55A00 or #FF8C00

Background (#212737):
- Good choice - not pure black, blue undertone adds depth
- Similar to slate/dark blue-gray popular in modern UIs

Text colors:
- #EAEDF3 - excellent primary text, soft white
- #C2C5CD - good muted secondary
