## What is Livo?

Livo is a PWA (Progressive Web App) that combines travel tracking with goal setting. Mark countries you've visited on an interactive map and track personal goals you've accomplished.

## Features

- **Interactive Map**: Click countries to mark them as visited or bucket list
- **Goals Tracker**: Add, complete, and track personal goals
- **Responsive**: Works on mobile and desktop as a PWA
- **Offline Support**: Data syncs when connection returns

## Tech Stack

Built with Next.js and Supabase for the backend. Uses react-simple-maps for the interactive map visualization.

## Why Supabase?

Supabase provides authentication, database, and realtime sync. Perfect for a personal project that needed user data without building a full backend.
